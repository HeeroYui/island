Installation
------------

Requirements: ``Python >= 2.7`` and ``pip``

Just run:

  pip install maestro

Install pip on debian/ubuntu:

  sudo apt-get install pip

Install pip on ARCH-linux:

  sudo pacman -S pip

Install pip on MacOs:

  sudo easy_install pip
